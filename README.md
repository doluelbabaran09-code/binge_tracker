# Binge Tracker (OOP Version)

This is a Python-based Binge Tracker application refactored to demonstrate the four core pillars of Object-Oriented Programming (OOP): Abstraction, Inheritance, Polymorphism, and Encapsulation.

### Project Details
- Language: Python 3.13
- Original Source: Refactored from my original procedural version of the Binge Tracker project.
